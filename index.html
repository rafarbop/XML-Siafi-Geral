Informações sobre arquivos XML e a Carga de Dados do SIAFI

O SIAFI possui um Subsistema de Importação e Exportação de Dados, onde é possível receber arquivos (Somente em formato XML - Extensible Markup Language) contendo as informações necessárias para executar funcionalidades de negócio do SIAFI, validar o formato dessas informações e encaminhá-las para o processamento.

Nessa forma de inclusão de dados, a geração de diversos documentos hábeis acontece de forma automática de acordo com as informações contidas nesse arquivo em formato XML que importamos para o sistema.

A Secretaria do Tesouro Nacional - STN - apresenta uma solução em Planilha de Excel para a conversão dos dados necessários para o formato XML - Link: SNT - Como incluir documentos por meio de processo Batch. Kolmar Kafran, servidor do IFPB, fez e disponibiliza uma versão modificada da planilha elaborada pela STN - Link: Planilha modificada de Kolmar Kafran

Um dos problemas da utilização dessa planilha é a necessidade do Microsoft Excel, sendo necessário a compra de uma licença, pois a versão gratuita online do Microsoft 365, não suporta a exportação em XML.

Dessa forma, devido a necessidade de utilização do processo de Carga de Dados do SIAFI, sem precisar adfquirir uma licença do Microsoft Excel e também com o objetivo de disponibilizar um alternativa mais simples, este site possibilita a inclusão de dados dos dados necessários a geração de um arquivo XML a ser importando no SIAFI.

Os dados incluídos não são enviados ou processados em qualquer outro site ou servidor, são processados pelo próprio navegador do usuário usando a linguagem de programação Python/Javascript e sendo disponibilizado o download do arquivo XML
